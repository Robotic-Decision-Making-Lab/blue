# Welcome!

Blue is an end-to-end ROS 2 framework designed to support developing, testing,
and deploying underwater robots. This site provides all of the information
that you need to get started with integrating Blue into your underwater robotics
project today!

:::tip Supported ROS 2 Versions

Blue provides support for ROS 2 **Humble**, **Iron**, and **Rolling**. You can
find documentation for each supported version on this site.

:::

## Main Features

Blue offers several features including:

1. A set of interfaces designed to enable development of custom localization,
   planning, and control algorithms alongside the existing
   interfaces/algorithms. You can learn more about the provided features here.
2. A collection of commonly-used underwater vehicle models like the
   [Blue Robotics](https://bluerobotics.com/) BlueROV2 and BlueROV2 Heavy.
3. ArduSub + Gazebo SITL integration to enable evaluating the performance of
   your algorithms in a simulated environment.
4. Docker integration to support deployment of your algorithms and
   implementation to hardware.

## Getting Help

We hope that you don't have any issues using Blue, but if you do, please check
